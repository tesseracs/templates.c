# Tesseracs — c template

Starter project for [Tesseracs](https://github.com/tesseracs) chat sessions.

Clone URL: `https://github.com/tesseracs/templates.c`

## Layout

- `main.c` — entry; includes `math_utils.h` and uses `add()`.
- `math_utils.h` / `math_utils.c` — prototype + implementation (both passed to `gcc` in `run.sh`).
- `run.sh` — builds and runs the executable.
